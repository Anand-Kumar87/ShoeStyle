# ShoeStyle - E-commerce Store

A modern, full-stack e-commerce application built with Next.js, TypeScript, Prisma, and Stripe.

## 🚀 Features

- **Product Management**: Browse, search, and filter products
- **Shopping Cart**: Add/remove items with persistent storage
- **Checkout**: Secure payment processing with Stripe
- **Authentication**: Sign in with email/password or Google OAuth
- **Admin Dashboard**: Manage products, orders, and coupons
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **SEO Optimized**: Server-side rendering and meta tags
- **Performance**: Image optimization and code splitting

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js
- **Payments**: Stripe
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Forms**: React Hook Form
- **Icons**: Lucide React
